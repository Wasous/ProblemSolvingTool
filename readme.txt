Backend (Node.js con Express):
- Express.js: Para manejar las rutas y las solicitudes HTTP.
- Sequelize: ORM para interactuar con PostgreSQL.
- PostgreSQL: Base de datos donde almacenas los usuarios.
- bcrypt.js: Para encriptar y comparar contraseñas.
- jsonwebtoken (JWT): Para manejar autenticación con tokens.
- dotenv: Para cargar variables de entorno desde un archivo .env.
